# VigramSDK Demo

VigramSDK Demo is a sample iOS application that shows how to integrate VigramSDK with a viDoc device.

Before running the demo, select your Apple Development Team and replace the Bundle Identifier with your own unique app identifier.

The demo includes:

- SDK authentication
- Bluetooth device scanning and connection
- Device status and configuration
- GNSS and NTRIP workflows
- Laser measurements
- PPK measurements
- Single point measurements
- Support log collection

## Requirements

- Xcode 16 or later
- iOS 15.5 or later
- VigramSDK 2.0.0-beta.3
- A valid VigramSDK token
- A compatible viDoc device

## Getting Started

1. Open `ViGRAM_SDK_demo.xcodeproj`.
2. Resolve Swift Package dependencies in Xcode.
3. Add your VigramSDK token in the demo app.
4. Build and run the app on a physical iOS device.
